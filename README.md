1인 쇼핑몰 프로젝트
# Description
● 개발 환경
  ○ IntelliJ 
  ○ HeidiSQL
  
● 사용 기술
  ○ Java
  ○ Spring Boot
  ○ Thymeleaf
  ○ JavaScript
  ○ Jquery
  ○ Ajax
  ○ JPA
  ○ MySQL
  
● 구현 기능
  ○ Spring Security를 이용한 회원가입, 로그인
  ○ 상품, 상품 옵션 등록
  ○ 고객센터 게시판, 댓글
  ○ 장바구니 (추가, 수량조절, 삭제)
  ○ KG이니시스 API를 이용한 결제 구현
  ○ 주문 내역
  ○ 리뷰 작성
