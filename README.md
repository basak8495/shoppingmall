# shoppingmall
1인 쇼핑몰 프로젝트
# Description
* 개발 환경  
  * IntelliJ  
  * HeidiSQL (로컬 환경)
  * MySQL Workbench (서버 배포 환경)
  
* 사용 기술  
  * Java  
  * Spring Boot  
  * Thymeleaf  
  * JavaScript  
  * Jquery  
  * Ajax  
  * JPA  
  * MySQL  
  
* 구현 기능  
  * 회원가입, 로그인  
  * 고객센터 게시판, 댓글  
  * 관리자 페이지 (옵션 추가, 상품 등록, 상품 옵션 등록, 주문 내역 관리)  
  * 장바구니 (추가, 수량조절, 삭제)  
  * KG이니시스 API를 이용한 결제 구현  
  * 주문 내역  
  * 리뷰 작성, 별점 등록  
  
* 서버 배포  
  * CloudType을 이용한 서버 배포 환경 구성  
  * Google cloud storage를 이용한 이미지 저장 공간 구성
